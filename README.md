# 貪吃蛇遊戲 🐍

一個使用HTML5 Canvas和JavaScript開發的經典貪吃蛇遊戲。

## 遊戲特色

- 🎮 流暢的遊戲體驗
- 🎨 現代化的UI設計
- 📊 記分系統與最高分記錄
- ⏸️ 暫停/繼續功能
- 🔄 遊戲重新開始
- 📱 響應式設計，支援手機和桌面

## 遊戲規則

- 使用方向鍵（↑ ↓ ← →）控制蛇的移動
- 蛇吃到紅色食物會變長並獲得10分
- 撞到牆壁或撞到自己身體時遊戲結束
- 按空白鍵可以暫停/繼續遊戲
- 遊戲結束後可以點擊「重新開始」按鈕

## 如何遊玩

1. 打開 `index.html` 文件
2. 使用方向鍵開始遊戲
3. 盡可能多地吃食物獲得高分！

## 技術特色

- **純JavaScript實現**：無需任何外部依賴
- **物件導向設計**：使用ES6 Class語法
- **本地儲存**：最高分會自動保存到瀏覽器
- **流暢動畫**：60FPS遊戲循環
- **碰撞檢測**：精確的碰撞判斷系統
